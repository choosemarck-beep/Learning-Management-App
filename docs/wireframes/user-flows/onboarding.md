# Onboarding User Flow

## Flow Overview

1. Welcome Screen → 2. Onboarding Steps → 3. Account Creation → 4. Dashboard

## Screen 1: Welcome Screen
- Pirate hat illustration
- "Ahoy Mate, Arrgh!" title
- Welcome message
- Skip button (left) | Next button (right)
- Progress indicators (dots)

## Screen 2: Onboarding Steps
- Feature highlights with illustrations
- Swipeable cards
- Continue button

## Screen 3: Account Creation
- Email/Password form
- Or social login options
- Terms & Privacy checkboxes
- Create Account button

## Screen 4: Dashboard
- User profile section
- Course cards
- Progress indicators
- Navigation menu

## Mobile Considerations
- Full-screen layouts
- Large touch targets (44px minimum)
- Swipe gestures for navigation
- Bottom navigation bar
